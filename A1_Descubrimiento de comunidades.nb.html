Descubrimiento de comunidades sin peso

En el mbito del analisis de redes biolgicas, el descubrimiento de comunidades permite agrupar los nodos aludiendo a funcionalidad comn o alto grado de relacin. En este Notebook se realiza un proceso de descubrimiento de comunidades para la red de oncogenes objeto de estudio, sin usar el peso de las aristas como elemento clave en el proceso.

Se carga el grafo

#Cargo datos
aristas <- read_excel("./Dataset1.xlsx", 
                    sheet = "WeigthedNetwork")
nodos <- read_excel("./Dataset1.xlsx", 
                    sheet = "NetworkNodes")
#Construyo el grafo
grafo_origen <- 
  graph_from_data_frame(d = aristas, vertices = nodos,  directed=F)

#Podo cogiendo como umbral 0.4
grafo <- delete_edges(grafo_origen, E(grafo_origen)[PESO<0.4])

Para el proceso de descubrimiento de comunidades se har uso de dos algoritmos, el primero de ellos basado en propagacin de etiquetas y el segundo, en optimizacin de la modularidad (ver Captulo II).

  • Deteccin de comunidades basada en propagacin de etiquetas.

Como paso previo se genera un vector con nmeros aleatorios que sirve para inicializar el algoritmo (si no, en cada ejecucin dar resultados diferentes).

set.seed(55)
inicio<-sample(c(0:176), size = 177, replace = TRUE)

Aplico el algoritmo y represento grficamente el resultado.

clp <- cluster_label_prop(grafo, initial = inicio)
plot(clp, grafo)
